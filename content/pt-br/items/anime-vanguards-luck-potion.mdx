## Entendendo o Papel dos Aumentos de Sorte no Anime Vanguards

Invocar a equipe perfeita é o cerne da progressão em *Anime Vanguards*, e todo jogador eventualmente procura uma vantagem para melhorar suas chances. Usar uma **poção de sorte do Anime Vanguards** é uma das maneiras mais populares de inclinar o RNG a seu favor, especialmente ao caçar aquelas unidades de nível alto tão elusivas. No entanto, entender exatamente o que uma **poção de sorte do Anime Vanguards** faz—e, mais importante, o que ela *não* faz—é essencial para economizar seus recursos e maximizar sua eficiência.

Muitos jogadores acreditam erroneamente que essas poções são uma solução para tudo em qualquer tipo de invocação. Na realidade, esses facilitadores são ferramentas especializadas que visam especificamente certas raridades, deixando outras intocadas. Ao aprender a mecânica por trás do sistema de invocação, você pode parar de desperdiçar seus recursos e começar a construir uma equipe de unidades mais poderosa.

## O que as Poções de Sorte Realmente Afetam?

Em *Anime Vanguards*, existem dois níveis principais de poções de sorte disponíveis: a poção Super Lucky e a poção Ultra Lucky. Ambos os itens são projetados para melhorar suas chances de obter unidades de maior qualidade nos vários banners do jogo.

De acordo com as mecânicas atuais do jogo, essas poções aumentam especificamente as taxas de drop para:
*   **Unidades Lendárias**
*   **Unidades Míticas**
*   **Unidades Exclusivas**

É vital notar que essas poções fornecem um bônus temporário, geralmente durando uma hora de jogabilidade ativa. Se você está planejando uma longa sessão de invocações, certifique-se de ter poções suficientes para cobrir todo o período para manter essas taxas otimizadas.

### O que as Poções de Sorte Não Influenciam

Um ponto comum de confusão entre novos jogadores é o escopo desses itens. Com base na [documentação oficial do Anime Vanguards](https://animevanguards.fandom.com/wiki/Summon), é importante esclarecer que **as poções de sorte não influenciam todas as raridades.**

| Raridade | Afetado por Poções de Sorte? |
| :--- | :--- |
| Rara | Não |
| Épica | Não |
| Lendária | **Sim** |
| Mítica/Exclusiva | **Sim** |
| Secreta | Não |
| Vanguarda | Não |

Além disso, relatos da comunidade indicam que essas poções não têm impacto na probabilidade de obter unidades "Shiny". Se você estiver caçando especificamente uma unidade Shiny ou uma unidade Vanguarda de alto nível, a poção de sorte padrão não fornecerá o benefício que você procura.

## Invocação Estratégica: Sistemas de Pity e Bônus

Um de tempos mais importantes do sistema de invocação de *Anime Vanguards* é a mecânica de "Pity". O Pity age como uma rede de segurança, garantindo uma unidade de uma raridade específica após você realizar um determinado número de invocações sem sucesso.

Um detalhe crucial para os jogadores lembrarem é que **as poções de sorte não reduzem sua contagem de pity.** O Pity é um sistema de progressão fixo que rastreia suas invocações totais, independentemente de quaisquer bônus ativos. Embora uma poção possa ajudá-lo a conseguir aquela unidade de raridade alta mais cedo, ela não acelera a contagem regressiva para uma invocação garantida.

### Limites de Pity por Tipo de Banner

Para planejar suas invocações com eficácia, consulte a tabela a seguir sobre invocações garantidas:

| Tipo de Banner | Pity Lendário | Pity Mítico | Pity Vanguarda |
| :--- | :--- | :--- | :--- |
| Especial / Seleção | 50 Invocações | 400 Invocações | 25.000 Invocações |
| Memória Especial | 50 Invocações | 400 Invocações | 20.000 Invocações |
| Banners de Evento | Varia | Varia | Único de 20k |

## Maximizando Sua Sessão de Invocação

Se você decidiu usar uma **poção de sorte do Anime Vanguards**, certifique-se de estar preparado para aproveitar ao máximo essa janela de uma hora. Como esses itens são valiosos, você não vai querer gastar metade do tempo do seu bônus limpando seu inventário ou gerenciando suas configurações.

### Lista de Preparação
1.  **Limpe Seu Inventário:** Certifique-se de ter espaço suficiente para lidar com um grande fluxo de unidades.
2.  **Ative a Venda Automática:** Vá para suas configurações e configure os recursos de Auto-Sell (Venda Automática). Isso evita que seu inventário fique cheio de unidades de menor raridade que você não precisa, o que, de outra forma, forçaria você a parar de invocar para limpar o espaço.
3.  **Verifique Seu Banner:** Verifique se o banner atual apresenta as unidades Míticas ou Lendárias específicas que você está buscando.
4.  **Confirme o Status VIP:** Se você tiver VIP, lembre-se de que seus custos de gemas para Banners Especiais são reduzidos, permitindo mais invocações por hora enquanto a poção de sorte estiver ativa.

## Mecânicas Avançadas: Unidades Shiny e Normalização

Embora a **poção de sorte do Anime Vanguards** não ajude com as taxas de Shiny, entender como gerenciar essas unidades ainda é essencial para o jogo em alto nível. Se você por acaso invocar uma unidade Shiny que não se encaixa na sua build atual, você não está preso a ela.

A Loja de Normalização (Normalize Shop), administrada pelo NPC Valentine, oferece dois serviços principais para unidades Shiny:
*   **Transferência de Shiny:** Permite que você mova o status Shiny de uma unidade para outra, desde que sejam do mesmo tipo. Isso custa 20.000 de Ouro e consome a unidade Shiny original.
*   **Normalização:** Remover a etiqueta "Shiny" de uma unidade concederá a você 5 rerolls de características. Esta é uma ótima maneira de reciclar unidades Shiny "inúteis" que você não pretende usar em sua equipe principal.

## Perguntas Frequentes

### Uma poção de sorte do Anime Vanguards aumenta minhas chances de obter unidades Secretas?
Não. De acordo com os dados internos do jogo, as poções de sorte afetam apenas as raridades Lendária, Mítica e Exclusiva. Elas têm impacto zero nas taxas de drop de unidades Secretas ou Vanguarda.

### Uma poção de sorte me ajudará a conseguir uma unidade Shiny?
Relatos da comunidade confirmam que as poções de sorte não influenciam a probabilidade de uma unidade ser Shiny. Você deve contar com sua sorte base ou com o gamepass Shiny Hunter para melhorar essas chances específicas.

### As poções de sorte acumulam?
O jogo geralmente trata isso como efeitos de status temporários. É melhor usar uma poção e esperar que a duração expire antes de consumir outra para garantir que você não esteja desperdiçando itens.

### A poção de sorte reduz o número de invocações necessárias para o pity?
Não, o sistema de pity é inteiramente separado dos bônus de sorte. A contagem de pity rastreia suas falhas totais em obter uma unidade de raridade alta e não é afetada por quaisquer itens ou bônus que você use durante o processo.
